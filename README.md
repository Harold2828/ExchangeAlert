### 1. Clone the Repository

```bash
git clone https://github.com/Harold2828/ExchangeAlert.git

```

### 2. Configure Environment Variables

Create a `.env` file in the root directory:

```ini
# Exchange Alert Configuration
EXCHANGERATE_API_KEY=your_api_key_here
EXCHANGERATE_BASE_CURRENCY=USD
EXCHANGERATE_TARGET_CURRENCY=COP
THRESHOLD=4200

# SMTP Configuration
EMAIL_HOST=smtp.gmail.com
EMAIL_PORT=587
EMAIL_SENDER=you@example.com
EMAIL_PASSWORD=your_app_password
EMAIL_RECIPIENT=recipient@example.com

# Runtime Configuration
RUN_TIME=09:00
```

> ğŸ” **Important:** Never commit this file. Use secure tools to manage secrets in production.
